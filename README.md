# TrafficAI-Router

A dependency-free Python shortest-path routing optimization layout engineered to balance city street congestion variables. By representing road intersections using a dense **2D Adjacency Matrix Graph**, this system enables automated smart city routing tools to adapt instantly to live telemetry metrics.

## 📊 Graph Architecture Specifications
* **Data Layout Strategy:** 2D Square Adjacency Matrix tracking connection paths.
* **Path Optimization Model:** Custom array-driven Dijkstra's path resolution algorithm.
* **Time Complexity:** Matrix traversal runs in clean $O(V^2)$ bounds, ideal for managing localized town sector grids.
